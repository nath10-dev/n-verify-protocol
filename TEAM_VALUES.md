# N-Verify Protocol - Team Values

## We Believe In

### 1. Ship Fast
- Iterate quickly
- Learn from users
- Don't perfect

### 2. User First
- Solve real problems
- Listen to feedback
- Deliver value

### 3. Be Transparent
- Open communication
- Share progress
- Admit mistakes

### 4. Stay Curious
- Keep learning
- Explore new ideas
- Question assumptions

### 5. Build Together
- Collaborate
- Support each other
- Celebrate wins
