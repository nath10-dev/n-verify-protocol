# N-Verify Protocol - User Journey

## 1. Sign Up
- Visit website
- Click Register
- Enter email/password
- Verify email
- Complete profile

## 2. Connect Wallet
- Click Connect Wallet
- Approve MetaMask
- Wallet connected

## 3. Submit Verification
- Click New Verification
- Select domain (Medical/Legal/Financial)
- Paste AI output
- Submit

## 4. Wait for Results
- Processing...
- Validators review
- Consensus reached

## 5. Get Certificate
- View results
- Download certificate
- Share verification
