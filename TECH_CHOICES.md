# N-Verify Protocol - Technology Choices

## Why This Stack?

### Frontend: Next.js 14
- Fast development
- Great performance
- Full-stack capabilities
- Large ecosystem

### Backend: Express + Node.js
- JavaScript everywhere
- Easy to learn
- Great for APIs
- NPM ecosystem

### Database: PostgreSQL
- Reliable
- Scalable
- SQL support
- JSON support

### Cache: Redis
- Fast
- Flexible
- Pub/sub support

### Blockchain: Ethereum
- Most secure
- Largest ecosystem
- Best tooling
