# N-Verify Protocol - Validator Guide

## What is a Validator?
A validator verifies AI reasoning chains and earns rewards.

## Requirements
- Stake tokens
- Domain expertise
- Reliable node

## Responsibilities
- Review submissions
- Validate reasoning
- Reach consensus

## Rewards
- Verification fees
- Token rewards
- Reputation points

## Getting Started
1. Register as validator
2. Stake tokens
3. Join validator network
4. Start validating
