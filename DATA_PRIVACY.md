# N-Verify Protocol - Data Privacy

## What We Collect
- Email and name
- Wallet address
- Verification data
- Usage analytics

## What We Don't Collect
- Private keys
- Passwords (hashed only)
- Sensitive personal data

## User Rights
- Access data
- Delete data
- Export data
- Opt-out of analytics

## Security
- Encryption at rest
- Encryption in transit
- Access controls
- Regular audits
