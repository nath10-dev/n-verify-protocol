# N-Verify Protocol - Git Branching

## Branch Strategy

### main
- Production code
- Only merged from develop

### develop
- Integration branch
- Feature branches merge here

### feature/* 
- New features
- Branch from develop

### bugfix/*
- Bug fixes
- Branch from develop

### hotfix/*
- Urgent fixes
- Branch from main

## Workflow
1. Create branch
2. Make changes
3. Pull request
4. Code review
5. Merge
